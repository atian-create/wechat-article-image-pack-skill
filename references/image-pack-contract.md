# Image Pack Contract

## 1. Article Visual Diagnosis

Name:

- article topic
- target reader
- core promise
- strongest emotion
- why the article needs images

## 2. Unified Visual Direction

Choose one:

- no-person knowledge infographic
- recurring creator character/IP
- product/workflow visual system
- clean editorial article cover
- hand-drawn explanation board
- dark technology grid
- warm journal knowledge cards

State:

- palette
- typography direction
- visual motif
- level of information density
- whether text should be rendered by model or added manually later

## 3. Cover Content Sheet

Include:

- main title: 6-12 Chinese characters when possible
- subtitle: 10-20 Chinese characters
- bottom line: one short sentence
- card labels: 2-4 short labels
- visual metaphor
- emotional hook

## 4. Cover Image Brief

Must include:

- aspect ratio
- main subject
- title placement
- supporting cards or labels
- background
- style
- avoid list

## 5. Three In-Article Images

Each image should match one article beat:

1. opening problem or scene
2. core method or turning point
3. final takeaway or next action

Each brief includes:

- image purpose
- short visible text
- main object or character
- layout
- style continuation

## 6. Generation Prompt

Prompts should be concrete:

- aspect ratio
- style
- subject
- visible text
- layout
- color
- texture
- negative instructions

Avoid:

- "make it beautiful"
- "high-end style" without specifics
- long article paragraphs inside the image
- random icons that do not support the content

## 7. Quality Checklist

Before final delivery, check:

- cover topic is recognizable in one second
- title is short enough
- image pack is visually consistent
- in-article images are simpler than the cover
- no private data or fake UI appears
- no broken text or distracting random symbols
- manual text overlay is recommended when model text may fail
